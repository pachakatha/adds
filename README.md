# LankaAds - Buy & Sell Anything in Sri Lanka

This is a high-quality, multi-page classifieds website built with HTML, JavaScript, and Tailwind CSS.

## Features
- **Modern Design**: Glassmorphism effects, smooth animations, and a vibrant color palette.
- **Posting Ads**: You can "post" ads (saved to your browser's local storage).
- **Photo Upload**: Supports uploading and previewing images for ads.
- **Filtering**: Filter ads by category (Vehicles, Electronics, etc.).
- **Responsive**: Works perfectly on mobile and desktop.

## How to Run
1. Simply double-click on `index.html` to open the website in your browser.
2. No installation or Node.js required! 
3. Ensure you are connected to the internet so the beautiful fonts and styling (Tailwind) can load.

## Files
- `index.html`: The home page.
- `listings.html`: The page where all ads are shown.
- `post-ad.html`: The form to sell an item.
- `ad-details.html`: The individual page for each item.
- `script.js`: Handles all the logic (saving ads, showing them).
- `styles.css`: Extra custom animations and styles.
